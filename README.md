# yhttp-boilerplate

A simple template for yhttp framework.


## Development


### Install prerequisites

```bash
sudo apt install \
  # Needs for compile Python extensions. \
  python3-dev \

  # For make virtual environment with Python. \
  python3-venv \

  # For database comunications. \ 
  libpq-dev
```


### Setup development environment

```bash
source activate.sh
```
